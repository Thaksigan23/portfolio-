## 🛠️ Installation & Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Thaksigan23/portfolio-.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd portfolio-
   ```

3. **Run a local server**:
   If you have VS Code, I recommend the **Live Server** extension. Alternatively, use `npx`:
   ```bash
   npx live-server
   ```
