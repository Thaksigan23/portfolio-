How I Built CareerGuide Hub

2026-06-10 · Projects

CareerGuide Hub started as a university project with a clear goal: help students connect their skills and interests to realistic career paths. I built it as a full-stack application and deployed it live on Vercel.

The problem

Many students know what they're studying but don't know which careers align with their strengths. I wanted a system where users could create a profile, list their skills, and receive structured career suggestions.

Tech stack

  • Frontend: React with component-based UI and protected routes
  • Backend: Node.js + Express REST API
  • Database: MySQL for user profiles and career data
  • Deployment: Vercel for the live demo

Key challenges

The hardest part was designing the recommendation flow—mapping free-form skill inputs to career categories without overcomplicating the UX. I solved this with a structured skill-selection form and a scoring algorithm on the backend that ranks careers by match percentage.

What I learned

  • How to structure a React app with authentication and dashboard views
  • Designing REST APIs that separate auth, profile, and recommendation concerns
  • Deploying a full-stack project and making it accessible via a live URL
